## 1. 本轮实验的硬性原则

1. 旧 checkpoint 和旧表格数字全部视为 legacy，不与新结果混合求均值。
2. 所有神经网络主结果必须跑满三个随机种子；均值和标准差使用总体标准差
   （`numpy.std`，`ddof=0`）。
3. checkpoint 只按验证集 Macro-F1 选择，测试集只在选定 checkpoint 后报告，不能用
   test F1 选择配置。
4. 英文主干使用 `bert-base-uncased`，中文主干使用 `bert-base-chinese`；BERT 基线为
   全参数微调。
5. 统一训练参数：AdamW、`lr=2e-5`、`weight_decay=0.01`、
   `label_smoothing=0.1`、`max_length=512`、`epochs=3`；PolitiFact batch=4，
   GossipCop/Weibo21 batch=16。
6. SocialDebias 统一结构为 `768→384→128→2`，不得把旧 `768→384→2` checkpoint
   混入新实验。
7. 正式主模型在三个数据集上统一使用 `surface_all`，不再按数据集挑选不同“最优”
   配置。其组成固定为：GRL + Consist + 8 维 NRC + InfoNCE + NLI 双软标签 +
   Adaptive λ。
8. `surface`、`surface_adaptive`、17 维特征以及去组件版本只用于消融和机制分析。
9. 自适应 λ 是否触发必须读取本轮同 seed BERT 的 `best_val_f1` 和实际训练集规模，
   禁止在论文中预先写死 PolitiFact“不触发”。
10. 结果方向与旧论文不一致时，以新结果为准，修改论点，不为复现旧数字而调参。

## 5. P2：重新训练基线

### 5.1 BiLSTM（PolitiFact、GossipCop）

```bash
for DS in politifact gossipcop; do
  for SEED in 42 2024 3407; do
    python scripts/train_lstm.py \
      --dataset "${DS}" --seed "${SEED}" \
      --epochs 3 --max_len 512 \
      --batch_size "$([ "${DS}" = politifact ] && echo 4 || echo 16)" \
      --lr 1e-3 --embed_dim 300 --hidden_dim 256 --num_layers 2 \
      --output_dir "${RUN_ROOT}/lstm" \
      2>&1 | tee "${LOG_DIR}/lstm_${DS}_seed${SEED}.log"
  done
done

python scripts/parse_lstm_results.py \
  --input_dir "${RUN_ROOT}/lstm" \
  --output "${RUN_ROOT}/lstm_summary.csv"

test "$(find "${RUN_ROOT}/lstm" -name 'lstm_*_result.json' | wc -l)" -eq 6 \
  && echo "BiLSTM 6/6 完成"
```

## 10. P7：六组件消融

完整模型加 6 个去组件变体，共 7 个配置。PolitiFact 和 GossipCop 各跑三 seed。
消融使用固定 λ，不启用 Adaptive λ，以保证每次只改变一个因素。

| 后缀 | 相对 Full 的唯一变化 |
|---|---|
| `abl_full` | 全组件 |
| `abl_no_grl` | `--lambda_bias 0` |
| `abl_no_infonce` | 移除 `--use_contrastive`，保留分类软标签 |
| `abl_no_consist` | `--lambda_consist 0` |
| `abl_no_surface` | `--surface_feat_dim 0`，不传 NRC |
| `abl_no_advaug` | 移除 paired 数据、InfoNCE、NLI 分类软标签 |
| `abl_no_labelsmooth` | `--label_smoothing 0 --lambda_fact_soft 0`，仍保留加权 InfoNCE |

以下循环已经把 7 个变体展开，并在每次训练后立即评估对应 checkpoint；不要再手工改
命令。若中途中断，可先查看已有 history/JSON，然后临时从 `CONFIGS` 删除已完成配置再跑。

```bash
CONFIGS=(full no_grl no_infonce no_consist no_surface no_advaug no_labelsmooth)

for DS in politifact gossipcop; do
  BS=$([ "${DS}" = politifact ] && echo 4 || echo 16)
  ORIG="data/sheepdog/news_articles/${DS}_train.pkl"
  ADV="data/qwen_adv/${DS}_p_entail_paper_v2.pkl"

  for CFG in "${CONFIGS[@]}"; do
    for SEED in 42 2024 3407; do
      LB=0.5
      LC=0.3
      SDIM=8
      LABEL_SMOOTH=0.1
      LEX_ARGS=(--surface_lexicon_path "${NRC_EN}")
      PAIR_ARGS=(--orig_pkl "${ORIG}" --adv_pkl "${ADV}")
      NLI_ARGS=(
        --use_contrastive --lambda_contrast 0.3 --temperature 0.07
        --use_soft_labels --alpha_floor 0.5
        --lambda_fact_soft 0.5 --soft_label_floor 0.5
      )

      case "${CFG}" in
        full) ;;
        no_grl) LB=0 ;;
        no_infonce)
          NLI_ARGS=(--lambda_fact_soft 0.5 --soft_label_floor 0.5)
          ;;
        no_consist) LC=0 ;;
        no_surface)
          SDIM=0
          LEX_ARGS=()
          ;;
        no_advaug)
          PAIR_ARGS=()
          NLI_ARGS=()
          ;;
        no_labelsmooth)
          LABEL_SMOOTH=0
          NLI_ARGS=(
            --use_contrastive --lambda_contrast 0.3 --temperature 0.07
            --use_soft_labels --alpha_floor 0.5
          )
          ;;
      esac

      SUFFIX="abl_${CFG}"
      python scripts/train_ablation.py \
        --dataset "${DS}" --language en --seed "${SEED}" \
        --epochs 3 --batch_size "${BS}" --max_length 512 \
        --lr 2e-5 --weight_decay 0.01 --label_smoothing "${LABEL_SMOOTH}" \
        --hidden_dim 384 --bottleneck_dim 128 \
        --lambda_fact 1.0 --lambda_bias "${LB}" --lambda_consist "${LC}" \
        --surface_feat_dim "${SDIM}" "${LEX_ARGS[@]}" \
        "${NLI_ARGS[@]}" "${PAIR_ARGS[@]}" \
        --save_dir "${MODEL_DIR}" --save_suffix "${SUFFIX}" \
        2>&1 | tee "${LOG_DIR}/${SUFFIX}_${DS}_seed${SEED}.log"

      python scripts/evaluate_ablation_adv.py \
        --dataset "${DS}" --language en --seed "${SEED}" \
        --save_suffix "${SUFFIX}" --variants A,B,C,D \
        --batch_size "${BS}" --max_length 512 \
        "${LEX_ARGS[@]}" \
        --ckpt "${MODEL_DIR}/socialdebias_${DS}_en_seed${SEED}_${SUFFIX}.pt" \
        --output_dir "${RUN_ROOT}/ablation_adv" \
        2>&1 | tee "${LOG_DIR}/${SUFFIX}_adv_${DS}_seed${SEED}.log"
    done
  done
done
```

## 12. P9：NLI 机制实验

旧论文对 `floor=0.5` 与 Soft14 的开关描述不够清楚。本轮采用可审计的两因素设计：

| 配置 | InfoNCE 的 NLI 权重 | 分类软标签 | floor |
|---|---:|---:|---:|
| `nli_hard` | 否 | 否 | — |
| `nli_contrast_only` | 是 | 否 | 0.5 |
| `nli_cls_only` | 否 | 是 | 0.5 |
| `nli_dual_floor0` | 是 | 是 | 0.0 |
| `nli_soft14` | 是 | 是 | 0.5 |

统一保留 paired 数据；`nli_hard` 保留普通 InfoNCE，只是不使用 `p_entail` 加权。五种配置
均在三个数据集、三个 seed 上运行，共 45 次。本阶段固定 λ，不传 `--adaptive_lambda`。

```bash
NLI_CONFIGS=(nli_hard nli_contrast_only nli_cls_only nli_dual_floor0 nli_soft14)

for DS in politifact gossipcop weibo21; do
  if [ "${DS}" = politifact ]; then
    BS=4; LANG=en; NRC="${NRC_EN}"
    DATA_ARGS=(--dataset politifact --language en)
    ORIG=data/sheepdog/news_articles/politifact_train.pkl
  elif [ "${DS}" = gossipcop ]; then
    BS=16; LANG=en; NRC="${NRC_EN}"
    DATA_ARGS=(--dataset gossipcop --language en)
    ORIG=data/sheepdog/news_articles/gossipcop_train.pkl
  else
    BS=16; LANG=zh; NRC="${NRC_ZH}"
    DATA_ARGS=(--use_weibo21)
    ORIG=data/weibo21_repo/data/train.pkl
  fi
  ADV="data/qwen_adv/${DS}_p_entail_paper_v2.pkl"

  for CFG in "${NLI_CONFIGS[@]}"; do
    case "${CFG}" in
      nli_hard)
        NLI_ARGS=(--use_contrastive --lambda_contrast 0.3 --temperature 0.07)
        ;;
      nli_contrast_only)
        NLI_ARGS=(
          --use_contrastive --lambda_contrast 0.3 --temperature 0.07
          --use_soft_labels --alpha_floor 0.5
        )
        ;;
      nli_cls_only)
        NLI_ARGS=(--lambda_fact_soft 0.5 --soft_label_floor 0.5)
        ;;
      nli_dual_floor0)
        NLI_ARGS=(
          --use_contrastive --lambda_contrast 0.3 --temperature 0.07
          --use_soft_labels --alpha_floor 0
          --lambda_fact_soft 0.5 --soft_label_floor 0
        )
        ;;
      nli_soft14)
        NLI_ARGS=(
          --use_contrastive --lambda_contrast 0.3 --temperature 0.07
          --use_soft_labels --alpha_floor 0.5
          --lambda_fact_soft 0.5 --soft_label_floor 0.5
        )
        ;;
    esac

    for SEED in 42 2024 3407; do
      python scripts/train_socialdebias_surface.py \
        "${DATA_ARGS[@]}" --seed "${SEED}" \
        --epochs 3 --batch_size "${BS}" --max_length 512 \
        --lr 2e-5 --weight_decay 0.01 --label_smoothing 0.1 \
        --hidden_dim 384 --bottleneck_dim 128 \
        --lambda_fact 1.0 --lambda_bias 0.5 --lambda_consist 0.3 \
        --surface_feat_dim 8 --surface_lexicon_path "${NRC}" \
        "${NLI_ARGS[@]}" \
        --orig_pkl "${ORIG}" --adv_pkl "${ADV}" \
        --save_dir "${MODEL_DIR}" --save_suffix "${CFG}" \
        2>&1 | tee "${LOG_DIR}/${CFG}_${DS}_seed${SEED}.log"
    done
  done
done

test "$(find "${MODEL_DIR}" -name '*_nli_*_history.json' | wc -l)" -eq 45 \
  && echo "NLI 机制实验 45/45 完成"

python scripts/aggregate_training_histories.py \
  --model_dir "${MODEL_DIR}" \
  --suffixes nli_hard nli_contrast_only nli_cls_only nli_dual_floor0 nli_soft14 \
  --expected_datasets politifact gossipcop weibo21 \
  --expected_seeds 42 2024 3407 --require_complete \
  --output "${RUN_ROOT}/nli_mechanism_summary.csv"
```
